# Amit Chandra - Portfolio

Passionate Python developer excelling in solving intricate problems and optimizing code, demonstrating a strong commitment to craftsmanship and efficiency in software development.

## About

This is a modern, space-themed portfolio website built with Next.js 14, showcasing my projects, skills, and experience as a Python Developer.

## Tech Stack

- **Framework:** Next.js 14 with App Router
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **3D Graphics:** Three.js with React Three Fiber
- **Language:** TypeScript

## Features

- 🚀 Space-themed design with animated backgrounds
- 💫 Smooth animations and transitions
- 📱 Fully responsive design
- 🎨 Modern UI with glassmorphism effects
- ⚡ Optimized performance

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
